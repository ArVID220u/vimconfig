# vimconfig
My vim configuration.

Containing a vimrc, and the contents of the .vim directory.

## Installation

- Clone this repository into the `~/.vim` directory:
```
git clone https://github.com/ArVID220u/vimconfig ~/.vim
```
- Copy the `vimrc` file to `~/.vimrc`.
```
cp ~/.vim/.vimrc ~/.vimrc
```
